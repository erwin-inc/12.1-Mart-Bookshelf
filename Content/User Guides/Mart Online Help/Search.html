Search

The contextual global search feature enables you to search across all the technical and business assets quickly and easily. The search results are displayed based on the search string.

To access search, on the menu, click .

The Search page appears.

Enter a search string in the Show me the data! box and then, click or press Enter. The search results display all the assets containing the search string.

From the search results page, you can also use the Filters section to view assets based on asset type and ratings.
