Best Practices to Add a Group User

The following are the best practices to add a Group User:

  • Suppose that you have a parent group that includes subgroups. To give access to all the subgroups under a parent group, add the parent group as a Group User.
  • Suppose that you want to give generic permissions to all users of a parent group and specific permissions to a subgroup. Add both, the parent group and the subgroup and give specific permissions to the subgroup. The permissions given to a subgroup take precedence over the permissions given to a parent group.
  • Suppose that you want to give generic permissions to a group and specific permissions to a particular user of the group. Add the group and add the particular user as a Windows user. The permissions given to a Windows user take precedence over the permissions given to a group user.
