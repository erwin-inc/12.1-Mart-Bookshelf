View NSM File Report

An NSM file can be applied to multiple models. You can view a report of models to which an NSM files is applied. Apart from models, this report displays the time stamp and editor.

To view NSM file report, follow these steps:

  1. In the NSM Files pane, select an NSM file.
  2. On the actions bar, click .
    A report with a list of models, and their modification time stamp opens.
