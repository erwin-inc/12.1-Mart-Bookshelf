erwin Mart Server

erwin Mart Server is one of the components of the erwin Data Modeling solution. It enables centralized model management, collaborative modeling with conflict resolution, and change management with auditing.

erwin Mart Administrator is the front-end web application for erwin Mart Server and enables communication between erwin Data Modeler and erwin Mart Server.

This guide walks you through the erwin Mart Administrator UI and explains its working.
