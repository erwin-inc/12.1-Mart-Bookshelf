User Interface

To get you started with using , this topic walks you through its UI, components, and their functions.

Once you have installed and configured , follow these steps to access and use it:

  1. Open .

    2. The Login page appears.

  2. Enter your erwin Mart Administrator credentials.
  3. Click Login.

After successful log in, the Home page appears.

UI Section

Function

1-Top Pane

Application Menu: Click this icon to expand the menu and access modules.

Toggle Light/Dark Theme: Click this icon to switch between dark and light modes of the UI.
REST Report: Click this icon to navigate to the REST reports feature.

You can generate REST reports only if you are a Server user.
Help: Click this icon to view the help topic corresponding to the current page.

User Account: Click this icon to manage your user account or log out.

Use this pane to access modules.

Use the buttons on this toolbar to perform tasks relevant to each module.
